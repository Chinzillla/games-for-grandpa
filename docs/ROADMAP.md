# Roadmap

- **v0.1:** Target Tap, Three in a Row, Paddle Rally prototype.
- **v0.2:** Duck Hunt and complete accessible UI redesign.
- **v0.3:** Full requested starter catalog: Duck Hunt, Tic Tac Toe, Connect Four, Pong,
  Space Defense, Maze Chase, Whack-a-Mole, Target Practice, Memory Cards, Jigsaw Puzzle,
  and Fishing.
- **v0.4:** Polish release for playtesting feedback: larger preview art, clearer audio
  controls, fishing balance, and optional photo jigsaw improvements.
- **v0.5:** Content expansion after usability testing, only if the base catalog remains
  comfortable for the intended player.

Each release must preserve mouse-only operation, original assets, large controls, and pure
domain models with focused algorithm documentation.

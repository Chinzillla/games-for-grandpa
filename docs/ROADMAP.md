# Roadmap

- **v0.1:** Target Tap, Three in a Row, Paddle Rally prototype
- **v0.2:** Duck Hunt and complete accessible UI redesign
- **v0.3:** Memory Match and Connect Four
- **v0.4:** original space-defense and maze-chase games
- **v0.5:** photo jigsaw and fishing

Each release must preserve mouse-only operation, original assets, large controls, and pure
domain models with focused algorithm documentation.

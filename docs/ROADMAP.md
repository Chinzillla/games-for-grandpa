# Roadmap

- **v0.1:** Target Tap, Three in a Row, Paddle Rally
- **v0.2:** Memory Match, Connect Four
- **v0.3:** original space-defense and maze-chase games
- **v0.4:** photo jigsaw and fishing
- **v0.5:** duck target gallery

Each release must preserve mouse-only operation, original assets, large controls, and pure
domain models with focused algorithm documentation.
